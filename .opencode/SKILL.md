# Habilidades del Proyecto

Este proyecto demuestra las siguientes habilidades y conocimientos adquiridos:

- **Flutter Avanzado**
  - Uso de arquitectura limpia (Clean Architecture)
  - Manejo avanzado de estado con `riverpod`
  - Integración de Firebase (Auth, Firestore, Storage)
  - Consumo de APIs públicas y privadas (REST)
  - Gestión de recursos multimedia (imágenes, galerías)

- **Backend & Persistencia**
  - Firebase Authentication (email/password)
  - Almacenamiento de imágenes en Firebase Storage
  - Consultas complejas a Firestore (filtros, ordenamiento)
  - Uso de modelos de dominio separados de la capa de datos

- **Integración de IA/ML**
  - Consumo de servicios de IA externos (API NSFW detection)
  - Validación de contenido antes de ser compartido

- **UI/UX Moderna**
  - Diseño responsivo y adaptable
  - Componentes reutilizables (widgets personalizados)
  - Transiciones fluidas y feedback visual
  - Galería de imágenes con scroll infinito y carga eficiente

- **Arquitectura Modular**
  - Separación clara de responsabilidades (modelo, dominio, data, usecases, presentación)
  - Código mantenible y testeable

- **Gestión de Proyectos**
  - Control de versiones con Git
  - Documentación interna del código y estructura del proyecto
